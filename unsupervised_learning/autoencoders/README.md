# Autoencoders Project

This project explores different types of autoencoders using TensorFlow Keras, including vanilla, sparse, convolutional, and variational autoencoders.

## Learning Objectives
- Understand what an autoencoder is
- Learn about latent space and bottlenecks
- Implement and compare vanilla, sparse, convolutional, and variational autoencoders
- Understand the Kullback-Leibler divergence and its use in generative models

## Tasks
1. Vanilla Autoencoder
2. Sparse Autoencoder
3. Convolutional Autoencoder
4. Variational Autoencoder
